About

The DPLA Similar Item Finder is an experiment in browsing and serendipitous discovery in the Digital Public Library of America. In other words, it’s a way to find interesting stuff in DPLA and explore connections between items from different places.

The website asks you for a DPLA identifier and a DPLA contributing hub. The Similar Item Finder looks for items from the selected hub that might have something in common with the item you gave it. It does this by randomly selecting one subject heading from your original item, and then searching the selected hub for that subject heading. It randomly selects 4 items from those search results to show you, and offers you a link to view the complete results set in DPLA.

A few tips for using the Similar Item Finder:

  • The ID for an item in DPLA can be found in the item's URL. It's the part after 'https://dp.la/item/'.
  • It only works if the item you provide has values in the Subject field.
  • The search results, though, aren’t limited to the subject field – it performs just a simple keyword search, to improve the chances of getting results.
  • After you get results for a search, you can refresh the page. That will cause the program to run again, selecting a different subject heading and presenting you newly selected results.
  • You have better chances of getting results if the record you provide has relatively simple subject headings, as opposed to long complex subject strings.

The DPLA Similar Item Finder was written in python and flask, and uses the DPLA API for interacting with the DPLA dataset. You can find the underlying code in github: https://github.com/elliotdwilliams/dpla-similar-item-finder This website is hosted on Heroku.

This project was created by Elliot Williams in 2022. It is not affiliated with DPLA, any contributing partner institution or hub, or anyone else. Questions or feedback, email elliotdwilliams@gmail.com, or find me on twitter.
